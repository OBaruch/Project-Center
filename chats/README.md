# Chat Summaries

Use this folder to store cleaned and structured summaries extracted from chats.

## Important rule

Do not blindly dump raw chat history into a public repository.

Instead:

- summarize,
- redact sensitive details,
- extract insights,
- convert ideas into structured Markdown.

## Recommended use cases

- decision histories,
- design discussions,
- project planning conversations,
- architecture reasoning,
- key prompts worth preserving.
