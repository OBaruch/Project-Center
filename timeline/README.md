# Timeline

Use this folder to document meaningful milestones over time.

Examples:

- major career changes,
- product launches,
- pivots,
- role transitions,
- learning phases,
- company or project milestones.

## Suggested use

You can create files such as:

- `2024.md`
- `2025.md`
- `career-milestones.md`
- `project-history.md`

Keep entries concise and factual.
