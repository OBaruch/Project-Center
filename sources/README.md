# Source Registry

Use this folder to document where project information came from.

Possible sources:

- ChatGPT
- GitHub repositories
- LinkedIn
- websites
- VPS notes
- cloud tools
- documents
- exported notes

## Suggested purpose

Track provenance for important projects so you know:

- where the information originated,
- when it was captured,
- whether it was summarized,
- and whether it is safe for public use.
